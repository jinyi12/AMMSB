# Writing Sources For This Skill

Use these sources in the following order of authority.

## 1. SIAM House Style

Primary authority for actual SIAM style questions:

- SIAM Style Manual

Use this source for:

- sectioning and presentation conventions
- copy-editing and house-style questions
- consistency of notation, punctuation, references, and formatting

## 2. Ordinary Scholarly Judgment

Primary authority for prose quality:

- clear mathematical exposition
- natural paragraph flow
- proportional claims
- distinction between what is defined, proved, implemented, and measured

This skill should not pretend that all prose judgments come from SIAM documents.

## 3. Higham's SIAM News Note

Useful only as a readability heuristic:

- avoid cliche
- avoid table-of-contents prose
- prefer readable, natural prose

Do not cite Higham's note as if it were a formal SIAM rulebook.

## 4. Related Writing References

Useful supplementary references for prose and mathematical writing:

- Nicholas Higham, *Handbook of Writing for the Mathematical Sciences*
- Donald Knuth, Tracy Larrabee, and Paul Roberts, *Mathematical Writing*
- Gopen and Swan, *The Science of Scientific Writing*
- Paul Constantine, “Three Resources for Math Writing”

Use these for broader writing craft, not for SIAM house-style authority.

## Practical Rule

When reviewing or rewriting:

- use SIAM documents for formal style claims
- use scholarly judgment for prose claims
- use Higham's note and related writing references only for light readability advice
