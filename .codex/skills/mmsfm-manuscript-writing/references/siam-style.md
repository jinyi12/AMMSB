# SIAM House Style

Use this file when the question is about actual SIAM style expectations rather than general prose quality.

## Scope

SIAM house style is the formal reference for submission and copy-editing conventions. It is the right source for questions about style requirements, notation formatting, references, sectioning conventions, and manuscript presentation.

## How To Use It In This Skill

- Treat the SIAM style manual as the authority on house style.
- Treat Higham's note only as a nonprescriptive source of readability advice.
- When reviewing prose, distinguish:
  `house style`,
  `scholarly readability`,
  `mathematical correctness`,
  `methodological accuracy`.

## Practical Consequence

When a user asks for a review or rewrite:

- Use SIAM style manual expectations for formal style questions.
- Use ordinary scholarly judgment for clarity and natural prose.
- Use Higham's note only to justify light observations such as:
  avoiding cliche,
  avoiding table-of-contents prose,
  and improving readability.
