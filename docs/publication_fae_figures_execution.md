# Publication Execution: FAE Autoencoder Figures

This document is the execution guide for the publication-quality FAE autoencoder figures generated by:

- `scripts/fae/fae_publication_figures.py`
- `scripts/fae/experiments/pipeline/publication_fae_figures.sh`

The runtime environment is `3MASB`.

## 1. Scope

This workflow covers the publication figures that belong to the FAE autoencoder study rather than the latent-geometry or latent-MSBM evaluation pipeline.

The FAE publication registry is the latent128 FiLM eight-run set:

- `fae_film_muon_l2_latent128`
- `fae_film_muon_ntk_99pct_latent128`
- `fae_film_muon_prior_latent128`
- `fae_film_muon_ntk_prior_latent128`
- `fae_film_adam_l2_latent128`
- `fae_film_adam_ntk_99pct_latent128`
- `fae_film_adam_prior_latent128`
- `fae_film_adam_ntk_prior_latent128`

Examples include:

- training and validation comparison figures
- held-out and train reconstruction field panels
- PSD spectral figures
- architecture / optimizer / denoiser comparison figures
- summary table export
- optional fig17 generated-field panel when an MSBM run is supplied

## 2. Command

```bash
cd /data1/jy384/research/MMSFM
conda run -n 3MASB \
  bash scripts/fae/experiments/pipeline/publication_fae_figures.sh
```

Useful overrides:

```bash
conda run -n 3MASB \
  bash scripts/fae/experiments/pipeline/publication_fae_figures.sh \
  --out_dir results/publication \
  --max_steps 50000 \
  --msbm_run_dir /data1/jy384/research/MMSFM/results/latent_msbm_muon_ntk_prior
```

Skip the optional MSBM-generated field figure:

```bash
conda run -n 3MASB \
  bash scripts/fae/experiments/pipeline/publication_fae_figures.sh \
  --no_msbm_fig17
```

Reevaluate missing FAE eval artifacts before plotting:

```bash
conda run -n 3MASB \
  bash scripts/fae/experiments/pipeline/publication_fae_figures.sh \
  --reevaluate_missing
```

## 3. Outputs

By default, figures are written to:

- `results/publication/`

Representative outputs include:

- `fig1_training_convergence`
- `fig4_held_out`
- `fig7_architecture_comparison_per_time`
- `fig10_sample0_train_fields`
- `fig10_sample0_held_out_fields`
- `fig12_psd_spectral_per_time`
- `fig12b_psd_mismatch_per_time`
- `fig16_sample0_all_times`
- `fig17_generated_fields`
- `fig8_summary_table`
- `table_summary.tex`

## 4. Notes

- This wrapper reruns the figure script directly. It is the correct path when you changed plotting code and want updated figures.
- `--reevaluate_missing` runs `scripts/fae/reevaluate_runs.py --missing_only` on the latent128 eight-run registry before plotting.
- The figure inventory and styling conventions are documented in [docs/publication_figures.md](publication_figures.md).
- The latent-geometry and latent-MSBM publication workflow remains separate and is documented in [docs/publication_latent128_msbm_execution.md](publication_latent128_msbm_execution.md).
